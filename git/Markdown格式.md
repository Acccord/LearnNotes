# Readme格式
<br>【<*br*>】换行<br>
# 【#】一级标题
## 【##】二级标题
### 【###】三级标题
#### 【####】四级标题
##### 【#####】五级标题
###### 【######】级别标题

``` text
1. 链接 :[Title](URL)
2. 加粗 :**Bold**
3. 斜体字 :*Italics*
4. 高亮 :==text==
5. 段落 : 段落之间空一行
6. 换行符 : 一行结束时输入两个空格
7. 列表 :* 添加星号成为一个新的列表项。
8. 引用 :> 引用内容
9. 内嵌代码 : `alert('Hello World');`
10. 画水平线 (HR) :--------
11. 复选框 :- [ ] -
12. 方框 : <kbd>?</kbd>
```
