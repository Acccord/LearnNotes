# Readme格式
<br>【<*br*>】换行<br>
# 【#】一级标题
## 【##】二级标题
### 【###】三级标题
#### 【####】四级标题
##### 【#####】五级标题
###### 【######】级别标题


- 【-】前面带点的行


> 【>】注释的行


- [ ] 【- [ ]】复选框


- [x] 【- [x]】选中的复选框

【**text**】**文字加粗**

- 链接 :[Title](URL)
- 加粗 :**Bold**
- 斜体字 :*Italics*
- *高亮 :==text==
- 段落 : 段落之间空一行
- 换行符 : 一行结束时输入两个空格
- 列表 :* 添加星号成为一个新的列表项。
- 引用 :> 引用内容
- 内嵌代码 : `alert('Hello World');`
- 画水平线 (HR) :--------
- 方框：- [ ] -
