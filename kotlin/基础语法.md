# - 1，可变长参数函数
> 函数的变长参数可以用 vararg 关键字进行标识：
```
fun vars(vararg v:Int){
    for(vt in v){
        print(vt)
    }
}
```

测试
```
fun main(args: Array<String>) {
    vars(1,2,3,4,5)  // 输出12345
}
```

## - 2，类型检测及自动类型转换

  if (obj is String) {
    // 做过类型判断以后，obj会被系统自动转换为String类型
    return obj.length 
  }
